1.前往HFS server下載getsql.sh
$ wget 120.96.143.10/get/getsql.sh


2.賦予執行權限，並執行它，取得mariadb的yaml檔
$ chmod +x getsql.sh
$ ./getsql.sh


3.建立pod
$ kubectl apply -f hdpsql/.


4.查看此pod所在的node位置
$ kubectl get pods -o wide


5.在ADM機器可以直接使用mariadb的client，進入mariadb server
$ mysql -uroot -proot -h ${node_ip}


(6.)在此叢集的pod，若有裝client端，可用 -h sqldb 直接互動
# mysql -uroot -proot -h sqldb
