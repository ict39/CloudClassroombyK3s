## 專題簡介
為減輕學校系統負擔使用由 Rancher 開源並獲 CNCF 認證的輕量化  K3S 建置雲端教室，採高可用性叢集讓機器發生故障時不會中斷其執行能力，我們可以在叢集建立學生上課要用的平台或軟體，例如: Hadoop、nginx，以及為了讓每個學生有專屬的運作空間，運用 Namespace 及 RBAC 兩大功能對整體資源和使用者權限做有效控管。
